# CEBICOR Contigo — Mi Espacio de Calma

Aplicación web progresiva (PWA) para el servicio de Psicología Corporativa de NEWLINK.

## Contenido
- Herramientas emocionales
- Kits por emoción
- Evaluaciones orientativas
- Biblioteca rápida
- Mi plan personal de bienestar
- Enlaces a Microsoft Bookings y WhatsApp
- Instalación en pantalla de inicio

## Publicación
Este repositorio está listo para conectarse con Netlify.

- Build command: dejar vacío
- Publish directory: `.`
- Branch: `main`

El archivo `index.html` está en la raíz del repositorio.
